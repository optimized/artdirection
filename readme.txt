== Description ==

"Per-post styles for new age art direction."

Pay your respects to print, get Art Direction.

This plugin allows you to have global archive and single page per-post styles and scripts. 

Need custom CSS everywhere? Try <a href="http://wordpress.org/extend/plugins/safecss/">WordPress.com Custom CSS</a>

Notes:
 * 0.2.4 Brings fixes for issues with caching plugins, support for localization, and the ability to put style-POSTID.css files in your template directory.
 * As of 0.2.1, global and single page custom code are now inserted in the <head>! Valid code for everyone!
 * 0.2 adds role checking and fixes a bug where duplicate global css can occur.

== Installation ==

1. Upload `art-direction` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Art Direction menu under the post entry text
